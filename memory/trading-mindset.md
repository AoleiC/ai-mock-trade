## 〇、仓位哲学：短线温度优先，大盘作共振

**核心认知：玩的是短线，短线情绪温度优先于大盘情绪温度。** 大盘温度是背景，短线温度才是当下能否动手的准绳。两者的组合决定仓位态度——短线好大盘好则共振可上仓位，短线好大盘坏则谨慎（局部机会），短线差大盘好也谨慎（防赚指数不赚钱），短线差大盘差则禁止（空仓）。

> **这条哲学 LLM 已落地，agent 只消费、不重算。** 大盘情绪（`temperature_score` / `temperature_label`）+ 短线情绪（`sentiment_analysis.sentiment_label`）已被 LLM 综合成 `final_analysis.sentiment_capital_alignment`（共振/警惕/撤退）与 `position_limit`。agent **不**自行对照"短线好/坏 × 大盘好/坏"四象限做本地判定，**直接采用** LLM 给出的 `alignment` 与 `position_limit`（策略 §1.1 / §1.2）。本哲学只是理解 LLM 输出的依据，不另立第四开仓条件。

## 一、个体纪律（硬红线）

### 1.1 不做的事

- **不打板** — 不追涨停，不在涨停板上买入
- **不接力** — 不赌连板，不在高位接力
- **不抄底** — 20 日线下方不接飞刀
- **不碰五类股** — 下跌趋势 / 心电图走势 / 无题材无业绩 / 财务造假 / 无放量涨停
- **不追撤退方向** — 已撤退方向个股红线 0%，无论涨多少不看不追
- **不买严重异动股** — `get_key_watch_stocks()` 返回的近 11 个交易日触发严重异动的个股，**一律禁买**。这类股短期累计涨幅已触发监管异动规则（如连续多日累计涨幅偏离值达标），存在核查停牌、特停、监管函等不确定风险，盈亏比极差。无论形态多好、资金多强、是否在活跃方向内，只要在禁买名单内就**不评估、不买入、不套利**（含已持仓股新进名单时不加仓，触发止损照常执行）

### 1.2 只做的事

- **首选活跃方向龙头** — 活跃方向内辨识度最高、带动性最强的核心个股
- **龙头无机会则套利** — 退而求其次做活跃方向内的中军或活跃股，活跃方向外有买点也可买（仓位适当收敛）
- **只做超预期** — 关键位置出现超出市场预期的表现时参与
- **只跟随趋势** — 20 日线上方、多头排列、量能充沛才进场
- **条件不够就空仓** — 空仓也是操作（具体判定见策略 §1.2 / §1.4）

### 1.2.1 龙头套利专项纪律（边界红线）

> **套利的本质**：龙头（涨停 / zdf>7% 高位 / 错过买点）时，资金溢出到同方向同板块中军 / 活跃股的跟随补涨机会。**不是**追高任何"长得像龙头"的股。

| 边界 | 红线 |
|------|------|
| **方向归属** | 必须在龙头所属的同一攻击方向内（同方向、同板块或主营细分重合）—— **不**做跨方向套利 |
| **资金面硬条件** | 主力净流入 > 0 **且** 换手 ≥ 5%（资金承接 + 活跃度双重确认）—— 主力流出标的**不**进套利评估 |
| **位置红线** | 个股 zdf ≤ 7%（满足 §2.2 高位纪律，"高位"定义见策略 §五）—— 龙头高位 ≠ 我也高位 |
| **买点类型** | 买点 2（分时承接超预期）/ 买点 3（突破超预期）—— **不**用买点 1（已过竞价窗口） |
| **不评估的标的** | 冷门跟风股（无独立逻辑、纯跟随脉冲）、跨方向套利、纯题材概念股 |
| **与低吸区龙头股冲突时** | 优先级：低吸区龙头股 > 强势可承接区套利 > 跟风股（不评估） |

**常见错误**：
- ❌ 龙头涨停 → 买同板块任何上涨的股（脉冲跟风）
- ❌ 龙头无买点 → 买"看起来像龙头"的强势股（已 zdf>7%，违反位置红线）
- ❌ 套利标的资金面流出（主力 -X 亿）仍买入（资金不承接 = 假套利）
- ❌ 把"中军"等同于"任何市值大的同板块股"（中军 = 板块第二梯队核心，有独立逻辑）

### 1.3 九条铁律

1. **不打板，不接力，只做龙头超预期。**
2. **首选活跃方向龙头，龙头不给机会再在活跃方向内套利，活跃方向外有买点也可轻仓。** 杂毛拉升是陷阱，看一眼都是错。**套利必须同方向同板块 + 资金面净流入 + zdf ≤ 7%，缺一不可。**
3. **跟着趋势走，20 日线是强弱分水岭。**
4. **五类股坚决不碰。**
5. **严重异动股不买。** `get_key_watch_stocks()` 名单内个股一律禁买（见 §1.1）。
6. **买卖别带感情，该卖就卖。** 犹豫 = 亏损扩大，止损是成本不是损失。
7. **主升浪只看成交量，放量滞涨就走。**
8. **盈利超 50%，先抽回本金，用利润滚。**
9. **只做市场跟随者，多头排列、量能充沛再进场，条件不够就空仓等。**

### 1.4 四条心得

- **只做新题材、只盯新龙头、只参与上升趋势。**
- **只选曾经有过涨停、股性活跃的标的。**
- **短线强势股核心特征：阳多阴少，阳线长、阴线短。**
- **买入后不盈利就走。** 买入第二天尾盘仍未盈利、或第三天还不盈利，即使尚未触及止损线，也必须主动评估是否提前离场——短线买的是"次日应有表现"的确定性预期，预期落空本身就是离场信号，不能等亏损扩大到止损线才被动执行（与策略 §2.3 时间止损同源，是其心法层面的提前预警）。
- **优先保住利润，非强势股少格局。** 走势与板块背离、放量滞涨、长上影即兑现落袋，不死扛等大涨——非强势股的"格局"是亏损的来源。加速止盈信号（策略 §2.3.1）触发即执行，把"等一等"的举证责任反转：出现异常默认倾向卖出，需明确理由才继续持有。
- **放量滞涨买卖对称，主力净流入≠必涨。** 铁律 7「放量滞涨就走」的判断同时适用于买入侧——分时已现放量滞涨 / 长上影的标的不买。主力大额净流入不是无条件加分项：大额资金进场却涨幅压在低位（资金量级与涨幅明显不匹配），是"钱进价不涨"的派发嫌疑，优先怀疑派发而非低吸（买入否决认知，落地判定见策略 §2.2.4 第 5/6 条、§四 checklist）。

## 二、仓位红线（个体级硬上限）

仓位红线是个体级硬约束，与 LLM 给出的账户级 `position_limit` 是**两个不同维度，分别生效**（叠加算式见策略 §1.1）：

| 维度 | 红线 |
|------|------|
| 单只个股 | 持仓 / 总资金 ≤ 30% |
| 同方向持仓 | 同方向持仓 / 总资金 ≤ 50% |
| 已撤退方向 | 红线 0% |

> **单只仓位统一上限 ≤ 30%**，具体多少由买点位置与追高风险自行研判，**不**按"是否落在 LLM 个股名单内/外"分档。

## 三、绝对红线（任何情况下不得违反）

| 红线 | 说明 |
|------|------|
| 不打板 | 不追涨停 |
| 不接力 | 不赌连板 |
| 不做撤退方向持仓 | 已撤退方向个股红线 0%（板块归属判定见策略 §1.3） |
| 不抄底下跌趋势 | 20 日线下方 |
| 不碰五类股 | 见 §1.1 |
| 不买严重异动股 | `get_key_watch_stocks()` 禁买名单内一律不买（见 §1.1） |
| 止损不犹豫 | 触发即执行（止损止盈阈值见策略 §2.3） |
| 不满仓赌博 | 单只 ≤ 30%，同方向 ≤ 50% |
| 条件不够就空仓 | 见策略 §1.2 / §1.4 |
| **套利不跨方向** | 套利标的必须在龙头所属同方向同板块，跨方向套利禁止 |
| **套利位置合规** | 套利标的 zdf ≤ 7% |
| **套利资金面合规** | 套利标的必须主力净流入 + 换手 ≥ 5%，资金不承接 = 假套利 |
